# cadetgui
Tinkering with cadet towards a streamlined cadet interface

# Instalation

## Pip with Cadet preinstalled

Create virtual environment
`python -m venv venv`
`source venv/bin/activate`

or alternatively via conda

`conda env create -f environment.yml`

## Activate venv
`conda venv activate cadet

or

`source venv/bin/activate`
## Install dependencies (use requirements.txt for exact versions)
`pip install -r requirements.txt`



## Install from copied repo

Using pip:

`pip install .`

Or editable mode for development:
`pip install -e .`

Alternative install from github

`pip install git+https://github.com/MoritzImendoerffer/cadetgui"`
